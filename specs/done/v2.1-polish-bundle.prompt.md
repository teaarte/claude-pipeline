3. `mcpSpawnRecorder` forwards `req.model` to `pipelineBeginAgent`.
4. `pipelineBeginAgent` already accepts `model` — no schema change needed.

**Tests** (`mcp/test/driver/core/fsm.test.ts` or `mcp/test/tools/begin-agent.test.ts` — extend existing):
